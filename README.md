# Лабораторна робота №3

**Виконала:** Соболева София К-24
**Варіант:** 24
**Схема:** №11

## Опис завдання
Програма реалізує багатопотокове виконання обчислень згідно з заданим орієнтованим графом залежностей (Схема №11). 
Використовується пул потоків фіксованого розміру (`nt = 7`), які вибирають задачі з черги по мірі виконання попередніх залежних дій.

**Вхідні дані варіанту:**
* Кількість потоків (`nt`): 7
* Кількості дій: a=6, b=9, c=7, d=7, e=8, f=6, g=5, h=7, i=9, j=8.

## Технічні вимоги та реалізація
* **Мова:** C++ (Стандарт **C++20**).
* **Синхронізація:** Використано тільки дозволені примітиви:
    * `std::binary_semaphore` (замість mutex для захисту черги).
    * `std::counting_semaphore` (для сигналізації потокам).
    * `std::latch` (для очікування завершення main).
    * `std::atomic` (для лічильників залежностей).
* **Вивід:** `std::osyncstream` для потокобезпечного виводу в консоль.
* **Обмеження:** `std::mutex` та `std::condition_variable` не використовувались.
